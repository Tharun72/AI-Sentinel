# AI Sentinel Architecture

## Vision
AI Sentinel is an AI-powered secure code auditor that analyzes source code, detects vulnerabilities, explains them using AI, suggests secure fixes, and verifies code integrity using blockchain.

## Workflow

User
↓
Upload / Paste Code
↓
Static Security Scan
↓
AI Vulnerability Analysis
↓
AI Secure Fix Generation
↓
Blockchain Verification
↓
Security Report
